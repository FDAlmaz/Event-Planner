Event Planner - это веб-приложение для организации и участия в мероприятиях, созданное на Django. Проект позволяет создавать события, регистрировать участников и управлять посещаемостью.

Ключевые возможности
- Управление мероприятиями: Полный цикл операций (CRUD) для создания, просмотра, редактирования и удаления событий.
- Система регистрации: Пользователи могут записываться на мероприятия с проверкой доступности мест.
- Личный кабинет: Страница "Мои события" с историей участия.
- Ограничение участников: Настройка максимального количества участников для каждого события.
- Адаптивный интерфейс: Удобный и современный веб-интерфейс.

Предварительные требования
Перед началом убедитесь, что у вас установлены:
- Python (версии 3.9 или выше)
- Менеджер пакетов Pip
- Git 

Запуск проекта:
Для запуска проекта требуется установленный Python версии 3.9 или выше, менеджер пакетов pip и Git.

После клонирования репозитория необходимо ввести в консоль pip install -r requirements.txt , python manage.py migrate,python manage.py createsuperuser и запустить сервер разработки командой python manage.py runserver.

После запуска приложение будет доступно в браузере по адресу http://127.0.0.1:8000
